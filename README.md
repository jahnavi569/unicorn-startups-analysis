# Indian-Unicorn-Startups-Analysis

SQL Analysis of Indian Unicorn Startups with Business Insights

## Project Overview

This SQL project analyzes Indian unicorn startups using data cleaning, feature engineering, advanced SQL queries, and business analytics to uncover trends in valuation, growth, profitability, market share, and startup performance.

## Key Insights

* Fintech and E-commerce industries contribute a major share of unicorn valuations.
* Several startups achieved unicorn status within a few years of founding, indicating rapid scalability.
* A small number of high-valued startups dominate the overall market valuation.
* Industry valuation distribution is highly concentrated across a few sectors.
* Startup valuation and profitability do not always move together.
* Custom Success Score identifies startups with strong valuation, growth speed, and acquisition performance.

1. Open MySQL Workbench.
2. Create the database and table structure.
3. Import the CSV dataset into the `unicorns` table.
4. Run `unicorn_analysis.sql`.
5. Explore the generated insights and results.
